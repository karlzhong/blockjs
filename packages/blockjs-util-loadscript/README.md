# blockjs-util-loadScript

## 简介
脚步异步加载函数

## 使用
```
# 全部安装blockjs
$ tnpm install @tencent/blockjs --save

# 只安装loadScript模块
$ tnpm install @tencent/blockjs-util-loadScript --save

# 引入
const loadScript = require('@tencent/blockjs-util-loadScript')

# use
loadScript('xxxx.js',function(){
    //callback
})
```
