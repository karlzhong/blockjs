# blockjs-code-base64

## 简介
base64编码

## Using npm:
```shell
# 全部安装blockjs
$ tnpm install @tencent/blockjs --save

# 只安装base64模块
$ tnpm install @tencent/blockjs-code-base64 --save
```

## 使用
```js


// 引入
const Base64 = require('@tencent/blockjs-code-base64')

// or
import Base64 from '@tencent/blockjs-code-base64'

// 样例
Base64.encode('xxx')
Base64.decode('xxx')

```
