# blockjs-bom-hash

## 简介
gethash

## Using npm:
```shell
// 加载整个blockjs
$ tnpm install @karlzhong/blockjs --save

// 或者单独加载子模块
$ tnpm install @karlzhong/blockjs-bom-hash --save
```

## 使用
```js

// 引入

const gethash = require('@karlzhong/blockjs-bom-hash')

// or

import gethash from '@karlzhong/blockjs-bom-hash'

// use
let hash = gethash('ad')

```
